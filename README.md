# AgLoop: Simple Basic Agent Loop

`agloop` implement a basic agent loop (or ralph loop) with jj as VCS (by default) and obsidian compatiable markdown for task tracking.

The whole codebase is hand-written and subject to human error :)

## Why?

Fun, I love JJ & markdown based task tracking.
